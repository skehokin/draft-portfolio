Saga - a blogging platform

Saga is a blogging platform deployed with Google App Engine and built primarily with Python. The user pages are dynamically generated with data from the Google Datastore database, which is requested via GQL, a variant of SQL.

This is then entered into a Jinja2 HTML template, which accesses CSS and JavaScript static files. User Authentication is fully functional, although not yet secured by SSL.
